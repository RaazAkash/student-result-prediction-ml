# Student Result Prediction using Machine Learning

This project predicts whether a student will Pass or Fail using Supervised Machine Learning.

## Problem Statement
The goal of this project is to build a machine learning model that predicts student results based on study behavior.

## Dataset
The dataset contains the following features:

- Study Hours
- Attendance
- Assignment Score
- Result (Pass/Fail)

Total Records: 1000

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Algorithm

Random Forest Classifier

## Project Workflow

1 Data Collection  
2 Data Preprocessing  
3 Train Test Split  
4 Model Training  
5 Prediction  
6 Model Evaluation  

## Model Accuracy

Model Accuracy: **98.5%**
